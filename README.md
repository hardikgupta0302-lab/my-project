# BuddyBot - AI Health Assistant Chatbot

BuddyBot is a web-based AI-powered chatbot that provides health-related tips and advice using OpenAI's GPT models.

## Features

✅ **AI-Powered Responses** - Uses OpenAI API for intelligent conversations
✅ **Health Tips** - Specialized in providing health and wellness advice
✅ **Web-Based Interface** - Modern, responsive UI
✅ **Real-time Chat** - Instant message responses
✅ **Easy Setup** - Simple configuration and deployment

## Requirements

- Python 3.7+
- OpenAI API Key (get it from https://platform.openai.com/api-keys)
